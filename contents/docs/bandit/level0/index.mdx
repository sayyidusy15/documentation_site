# Bandit Level 0

## Level Goal
Level ini bertujuan untuk membuat Anda terbiasa dengan cara melakukan koneksi ke game menggunakan SSH. Jika Anda belum pernah menggunakan SSH sebelumnya, artikel ini akan membantu Anda memulai.

## Level Information
- **Host**: bandit.labs.overthewire.org
- **Port**: 2220
- **Username**: bandit0
- **Password**: bandit0

## Walkthrough

1. Untuk memulai Level 0, Anda perlu melakukan koneksi ke server game menggunakan SSH. Gunakan perintah berikut:

```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
```
